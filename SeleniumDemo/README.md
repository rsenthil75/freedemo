## Steps to install HomeBrew and ChromeDriver on Mac

Steps:

	1. Install HomeBrew
	
```
	/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
	
	2. Install ChromeDriver
	
```
	brew install -cask chromedriver
```
